# Serandib Journeys

Marketing website for **Serandib Journeys**, a Sri Lanka travel agency — private tours, wildlife safaris, cultural journeys and custom itineraries.

## Tech stack

- [Vite](https://vitejs.dev/) + [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) (centralized teal/gold brand theme)
- [React Router](https://reactrouter.com/), [Framer Motion](https://www.framer.com/motion/), [Leaflet](https://leafletjs.com/)

## Getting started

```bash
npm install
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # production build → dist/
npm run preview  # preview the production build
npm run lint     # eslint
```

## Deployment

Deployed on [Vercel](https://vercel.com/). Framework preset: **Vite** (build command `vite build`, output directory `dist`).
